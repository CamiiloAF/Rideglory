# Rideglory — Guía de marca (beta)

## Paleta

| Rol | Hex | Uso |
|-----|-----|-----|
| Primario | `#f98c1f` | Acento, símbolo del logo, CTAs |
| Primario oscuro | `#e07510` | Sombras del gradiente, estados pressed |
| Primario claro | `#fbab56` | Brillo del gradiente, highlights |
| Fondo base | `#0D0D0F` | Fondo de icono, dark UI |
| Superficie | `#1A1A1F` | Tarjetas, contenedores |
| Borde | `#3D2810` | Bordes sutiles sobre dark |
| Texto claro | `#F1F5F9` | Texto principal sobre dark |
| Texto secundario | `#94A3B8` | Subtítulos, taglines |

**Gradiente del símbolo:** lineal `#fbab56` → `#f98c1f` → `#e07510`.

## Regla de oro (del design system)
Sobre el naranja, texto/iconos/elementos van **oscuros** (`#0D0D0F`), nunca blanco.

## Tipografía
- **Space Grotesk** — wordmark y títulos (Bold 700).
- Tagline en mayúsculas con tracking amplio (letter-spacing ~14).

## Wordmark
`Ride` en blanco + `glory` en naranja → "Rideglory". Tagline: **COMUNIDAD MOTERA**.

## Conceptos generados
1. **Monograma "R"** (bold / línea / badge) — más versátil como icono de app.
2. **Casco motero** — máximo reconocimiento de categoría.
3. **Moto + ruta** — enfatiza rodada y tracking en vivo.
4. **Emblema / parche** — sentido de comunidad/crew.
